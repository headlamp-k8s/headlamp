# headlamp

![Version: 0.7.0](https://img.shields.io/badge/Version-0.7.0-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: 0.11.1](https://img.shields.io/badge/AppVersion-0.11.1-informational?style=flat-square)

Headlamp is an easy-to-use and extensible Kubernetes web UI.

**Homepage:** <https://github.com/headlamp-k8s/headlamp/tree/main/charts/headlamp>

## TL;DR

```console
$ helm repo add headlamp https://headlamp-k8s.github.io/headlamp/
$ helm install my-headlamp headlamp/headlamp --namespace kube-system
```
